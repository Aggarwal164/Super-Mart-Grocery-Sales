# Super Mart Grocery Sales Analysis
This dataset contains retail sales data from a supermarket (Supermart Grocery), where each row represents an individual customer order.
It is designed for analyzing sales performance, profit trends, and customer purchasing patterns over time.

The dataset includes key business variables such as:

.Order Date → date when the order was placed
.Sales → total revenue generated from the order
.Profit → profit earned from the transaction

From the dataset, additional features were derived:

.Order Day, Order Month, Order Year → extracted from the order date for time-based analysis
.Profit_Margin → calculated as Profit / Sales to evaluate profitability
